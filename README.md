# motion-pool

A pool for RubyMotion.

## Installation

Add this line to your application's Gemfile:

    gem 'motion-pool'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install motion-pool

## Test

	$ PLATFORM=ios rake spec output=colorized2
